# Coast Analytics

Predictive CRM intelligence and test/learn platform for multi-property
hospitality operators. The flagship AI tool of [Oak Island AI](https://oak-island-ai.vercel.app/consultation).

**Production:** [predictive.oakislandmarketing.com](https://predictive.oakislandmarketing.com)

A sister product to Pinelands Predictive (the cannabis-retail variant of the
same engine). Logic is shared, the brand and synthetic dataset are
hospitality-specific.

## What is in here

- **client/** Vite + React marketing site and `/demo` workspace. Tailwind on
  the Oak Island AI palette, Poppins typography.
- **server/** Express API (Node 18+, ESM). Serves the 13 predictive models
  as JSON, handles the demo-gate Klaviyo integration, and writes leads to a
  small SQLite store with a /tmp fallback for Vercel.
- **ml-service/** Optional Python (Flask + sklearn) ML service. Local-only,
  not required, JS heuristics are the durable default.
- **data/** Synthetic Northwood Hospitality Group dataset, regenerated by
  `npm run seed`.
- **api/** Vercel serverless entrypoint that wraps the Express app.

## Quickstart

```bash
git clone https://github.com/bretthahn88/coast-analytics.git
cd coast-analytics
npm install
npm install --workspace=client
npm install --workspace=server
cp .env.example .env       # fill in KLAVIYO_API_KEY if you want the gate live
npm run seed               # generates data/*.json and data/*.csv
npm run dev                # starts both client (5173) and server (3001)
```

Open http://localhost:5173.

## Northwood Hospitality Group (the fictional demo client)

Nine fictional properties in the Carolina coast, generated synthetically:

- **Wrightsville Beach Resort** flagship hotel, highest volume
- **Bald Head Island Club** luxury resort, highest avg folio value
- **Cape Fear Inn** boutique hotel
- **Brunswick Harbor Hotel** boutique hotel
- **Kure Beach Grill** fine dining restaurant
- **Carolina Beach Tavern** casual restaurant
- **Southport Waterfront Lodge** spa
- **Holden Beach Pavilion** event venue
- **Oak Island Pier House** boutique inn

Loyalty program is **Northwood Rewards** with tiers **Silver**, **Gold**,
**Platinum**. Seasonality is summer-peak (Jun through Aug), with a holiday
bump in late December and shoulder dips in November and March through April.

## The 13 models

Core: Churn Risk, Revenue/Folio Forecasting, Tier Migration, Campaign Response.

Guest Scoring: Decliner Detection, Rising Stars, Near Platinum, VIP Likely,
Guest Lifetime Value.

Basket and Cross-Property: Basket Recommendation, Upsell, Cross-Sell,
Return-Visit Predictor.

See `client/src/demo/models.config.js` and `server/src/ml-models.js` for the
canonical IDs.

## Vercel deployment

The repo is Vercel-ready. `vercel.json` builds the client and wires every
`/api/...` request through `api/index.js` to the Express app. Set the
environment variables from `.env.example` in Project > Settings > Environment
Variables.

The Vercel project is configured with `predictive.oakislandmarketing.com`
as its production domain. CNAME the subdomain at the registrar to whatever
Vercel surfaces in the Domains tab (typically `cname.vercel-dns.com`).
